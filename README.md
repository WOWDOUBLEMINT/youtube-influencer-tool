# YouTube 邮箱线索挖掘工具
基于 Node.js + YouTube Data API v3 开发，用于关键词搜索 YouTube 视频/创作者，快速挖掘公开邮箱线索，支持缓存、筛选、导出、中英文切换。

## ✨ 功能特性
- 核心功能：关键词搜索 + 创作者邮箱展示
- 搜索缓存快照，避免重复请求
- 一键筛选「仅含邮箱」的有效线索
- 数据统计（总结果数 / 有效邮箱数）
- Excel 批量导出（支持仅导出有效线索）
- 中英文界面切换
- 优化列表展示：视频标题、链接、频道主页、粉丝数等

## 📌 重要说明
- 本工具依赖 **YouTube Data API v3**，**必须自行生成 API Key**
- 受 API 配额限制，**单日搜索次数约 15 次**
- 建议关键词完整、准确后再搜索，避免浪费配额

## 🚀 本地运行（基于 Node.js）
### 1. 环境要求
- 已安装 **Node.js**

### 2. 安装依赖
```bash
npm install
```

### 3. 配置 API Key
在项目配置文件中填入**你自己生成的 YouTube Data API v3 Key**
（API Key 需要去 Google Cloud Console 申请并开启对应 API）

### 4. 启动项目
```bash
npm start
```
启动成功后，在浏览器打开对应地址即可使用。

## 💡 使用建议
- 搜索前确认关键词完整准确，减少无效调用
- 优先使用「仅显示含邮箱」筛选，提升效率
- 导出结果可直接用于后续业务跟进
